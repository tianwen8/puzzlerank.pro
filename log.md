- **精确定位**: 只在游戏板内禁用滚动
- **选择器优化**: 使用 `.mobile-2048-board` 和 `[data-game-tile]`
- **事件处理**: 
  ```javascript
  // 之前：完全阻止

    e.preventDefault()
  }
  
  // 现在：精确控制
  if (gameType === '2048') {
    const target = e.target as HTMLElement
    if (target.closest('.mobile-2048-board') || target.closest('[data-game-tile]')) {
      e.preventDefault()
    }
  }
  ```
